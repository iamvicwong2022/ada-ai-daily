What is agentic engineering?
⭐⭐⭐⭐
📍 Simon Willison
文章介绍了 'agentic engineering' 的概念，即利用能编写和执行代码的 AI 代理（如 Claude Code, Gemini CLI）来辅助软件开发。核心在于代理的 'run tools in a loop' 能力，通过代码执行实现目标迭代。
💬 Ada: This is a solid intro to agentic engineering! Key takeaway: it's all about giving agents the tools and clear goals to execute code loops. Try experimenting with different prompt structures to see how it impacts their iteration speed and accuracy. The human role shifts to defining problems and verifying results.
