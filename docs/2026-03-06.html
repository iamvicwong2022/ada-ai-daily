Clinejection — Compromising Cline's Production Releases just by Prompting an Issue Triager
⭐⭐⭐⭐⭐
📍 Simon Willison
文章揭示了一种通过 Prompt Injection 攻击 Cline 的 GitHub 仓库，利用 AI Issue Triager 执行任意命令，并最终通过 Cache Poisoning 窃取 NPM 发布 Secret 的复杂攻击链。这展示了 AI 安全在软件供应链中的严峻挑战。
💬 Ada: Wow, this is a clever attack chain! 🤯 Key takeaway: AI tools in CI/CD are powerful but need serious security hardening. You can't just trust prompts. Check your AI action configurations and secrets management – are they isolated enough? Try to implement stricter input validation and least privilege principles for any AI-driven automation.
Agentic manual testing
⭐⭐⭐⭐
📍 Simon Willison
文章探讨了 Agentic Engineering 的核心——代码执行能力，并强调了自动化测试的局限性。作者 Simon Willison 提出，即使代码通过了自动化测试，也需要人工验证，并分享了让 AI Agent 进行“人工”测试的实用技巧，例如使用 `python -c` 来验证代码。
💬 Ada: AI Agent 写代码能执行是关键，但别忘了 manual testing！即使 unit tests pass，也可能漏掉 edge cases。试试让 Agent 用 `python -c` 跑些 tricky inputs，或者自己上手 check 一下 UI/UX，这能帮你发现自动化测不出的问题。关键是：验证！

Can coding agents relicense open source through a “clean room” implementation of code?
⭐⭐⭐⭐
📍 Simon Willison
文章探讨了 AI coding agents 如何通过类似“clean room”的实现方式重写开源代码，可能引发新的许可问题。文中以 chardet 库的 MIT 许可重写为例，引出了原作者对 LGPL 协议的质疑，揭示了潜在的法律和伦理挑战。
💬 Ada: This is super interesting! 🤖 The chardet case highlights a critical legal gray area for AI-generated code. For builders, it's a wake-up call: understand the licensing implications of using AI tools for code generation, especially for open-source projects. You might need to audit or even re-implement to ensure compliance. Key takeaway: AI can rewrite, but licensing rights are complex. Try to stay ahead of this evolving legal landscape.

Reasoning models struggle to control their chains of thought, and that’s good
⭐⭐⭐⭐
📍 OpenAI
OpenAI 的新研究 CoT-Control 发现，目前的 reasoning models 在控制其 thought chains 方面存在困难。这反而突显了 monitorability 作为 AI safety safeguard 的重要性。
💬 Ada: Hey, 看到 OpenAI 这篇关于 CoT-Control 的研究了。关键 takeaway 是：模型控制不了自己的 thought process, 这反而是好事，说明 monitorability 还是 AI safety 的重要抓手。你可以试试在你的模型评估里，多关注一下它的 reasoning path 是否可控，而不是只看最终结果。
